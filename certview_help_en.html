jsrsasign Certifificate Viewer is to see a X.509 digital certificate on your web browser based on jsrsasign pure JavaScript PKI and crypto library.

View Certificate

jsrsasign Certifificate Viewer has multiple ways to view certificates.

via textbox with PEM certificate

Paste a PEM certificate string to the textbox and click View button.

via textbox with FQDN

Type FQDN domain name in the textbox and click View button.

via File Open Dialog

Click Choose File button, choose PEM oer DER binary certificate file then click View button.
